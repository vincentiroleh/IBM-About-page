# [IBM About us page - Africa](https://developer.ibm.com/africa/aboutibm/)

A clone of the IBM About Us Page - Africa

### Demo

Demo is hosted at [LINK](https://hello-node-turbulent-pangolin.eu-gb.mybluemix.net/)

## Technologies

- HTML
- CSS
- [NodeJS](https://nodejs.org/) - Runtime Environment
- [ExpressJs](https://expressjs.com/) - Web Application Framework

### Installation

- Install [NodeJS](https://nodejs.org/) on your computer
- Clone this repository using `git clone https://github.com/vincentiroleh/IBM-About-page`
- Change director `cd BM-About-page`
- Run `npm install` to install all dependencies
- Run `npm start` to start the server
- Navigate to [localhost:3000](http://localhost:3000/) in browser to access the application
